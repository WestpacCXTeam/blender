Installing your Blend

  1. On the very top: Conditional <html> tag with "no-js" class.
    This is so our CSS can select older IE versions. The "no-js" class is replaced with "js" as soon as js runs.

  2. Inside the header: Two meta tags.
    Those are for encoding and viewport sizing.

    3. <% if( _hasSVG ) { %>

  3. Inside the header: The grunticon stylesheet loader.
    To allow us to SVGs in the GUI we need some smarts to load the appropriate stylesheet. To use your icons now include them like so: <span class="icon icon-size-sm icon-face-happy"></span> Read more about this technique.

    4. <% } %>

  4. Inside the header: A minified inline script.
    We remove the aformentioned "no-js" class here to avoid any flashes of styling.

  5. Inside the header: The gui.min.css file.
    Your GUI-blend styles.

  6. Inside the header: Conditional scripts for older IE.
    These scripts are important to enable responsive layouts and HTML5 elements in IE8.

    7. <% if( _hasJS ) { %>

  7. Right after the opening <body> tag: A conditional noscript block.
    This informs screenreaders and visitors in general that javaScript is turned off and that it is needed for this page.

  8. Before the closing </body> tag: The gui.min.js file.
    Your GUI-blend script.

    9. <% } %>

Changing your Blend

You can change your blend any time you like. Just go to the Blender, add or remove elements and re-blend another selection.
